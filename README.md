# Beauty Salon Website

A clean and responsive Beauty Salon website built using **HTML, CSS, and JavaScript**.
The project showcases a simple UI, smooth navigation, and a functional booking form.

## Features

* Responsive layout with smooth navigation
* Hero section with call-to-action
* Service cards with modern styling
* Appointment booking form
* Contact information section
* Clean and minimal UI design

## Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling and layout
* **JavaScript** – Form handling and interactions
